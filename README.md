# Eye Training System — OTA Update Repository

This repository hosts firmware binaries for over-the-air updates to Eye Training System devices.

## For Device Users

Your device checks this repository automatically for firmware updates. When an update is available, you'll see a notification on your device's LCD screen.

To update:
1. Make sure your device is connected to WiFi (internet access required)
2. Go to **Settings > Online Update** on your device
3. Press OK to check for updates
4. If an update is available, press OK to download and install

## Repository Structure

```
firmware.json    - Update manifest (version, download URL, checksums)
firmware/        - Compiled firmware binaries
LICENSE          - Usage terms
```

## Current Version

See `firmware.json` for the latest available firmware version.
